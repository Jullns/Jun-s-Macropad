## BOM:
- Seeed XIAO RP2040
- EC11 Rotary encoder
- MX-Style switches (x8)
- Blank DSA keycaps (x8)
- SK6812 MINI-E LEDs (x2)
- 4x M3x16 Bolt
- 4x M3 Heatset

## Challenges:

In this project, I learned a lot about using KiCad and Fusion, finally bringing the Macropad project to life. My biggest challenge was Fusion, as I had never used 3D design software before. I had to follow many steps to create the case; however, by following tutorials and doing research, I successfully completed the project and gained a great deal of knowledge.
